# Rupali Construction
A modern, ultra-premium real estate and construction portfolio built with React, Vite, and Tailwind CSS.

## Features
- **Dynamic Portfolio Grid**: Interactive filters for Residential, Commercial, and Renovation projects.
- **Project Detail Page**: Deep dive pages featuring split-image galleries, quick stats, dynamic cost breakdowns, and engaging CSS/SVG animations.
- **Skeleton Image Loaders**: Smooth, asynchronous image loading with pulse animations.
- **Modern Animations**: Framer Motion for scroll-reveals, smooth page transitions, and SVG line-art drawings.
- **Full Custom Styling**: Ultra-clean UI utilizing Tailwind CSS for glassmorphism, precise gradients, and typography.

## Quick Start
1. Run `npm install`
2. Run `npm run dev`
3. Enjoy!
